    DetailEditFormController Class

    Extends EditFormController
    Defined in: addon/controllers/detail-edit-form.js:9
    Module: ember-flexberry

    Controller for the Detail's Edit Forms. It lets to interact between agregator's and details's forms.

    This class re-exports to the application as /controllers/detail-edit-form. So, you can inherit from ./detail-edit-form, even if file app/controllers/detail-edit-form.js is not presented in the application.

    Example:

    // app/controllers/order.js
import DetailEditFormController from './detail-edit-form';
export default DetailEditFormController.extend({
});

Then proper request to load data is formed (it considers current page, filter, etc). After the data loading data are displayed on lookup modal window.         Number of records to display on page.

      • [page] String optional

        Current page to display on lookup window.

        Current sorting.

        Current filter.

        Current limit predicate.

    Handles action from lookup choose action. It opens modal window where availible values are shown. In order to customize content of all lookup modal window there is such a way: 1.
